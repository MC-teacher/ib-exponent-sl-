SECTION 01

지수함수

y = aˣ (a > 0, a ≠ 1) — 뜻, 그래프, 성질

정의

지수함수의 뜻

a > 0, a ≠ 1 일 때, 실수 전체의 집합을 정의역으로 하는 함수 y = aˣ 를 a를 밑으로 하는 지수함수라 한다.
  • 1정의역: 실수 전체 / 치역: 양의 실수 전체
  • 2일대일함수
  • 3a>1 → x 증가 시 y 증가
    0<a<1 → x 증가 시 y 감소
  • 4점 (0, 1) 통과
    점근선: x축 (y = 0)
TOK

a=1이면 y=1 (상수함수). 왜 a≠1 이어야 하는가? 수학에서 "흥미로운 함수"의 조건은?

그래프 비교

a > 1 vs 0 < a < 1

항목a > 1 예: y = 2ˣ0 < a < 1 예: y = (½)ˣ
증감증가감소
기준점(0, 1) 통과
점근선x축 (y = 0)
치역양의 실수 전체
x → +∞y → +∞y → 0
x → −∞y → 0y → +∞

두 그래프는 y축에 대해 대칭  |  y=(½)ˣ = 2⁻ˣ

그래프 변환

지수함수의
그래프 변환

y = f(x) → y = f(x - a) + b
x축으로 a, y축으로 b 평행이동
y = 2^(x+1) − 3

← y = 2ˣ를 x축으로 −1, y축으로 −3 이동
점근선: y = −3

y = (⅓)^(x−1) + 2

← y = (⅓)ˣ를 x축으로 +1, y축으로 +2 이동
점근선: y = 2

TOK

그래프가 x축에 "한없이 가까워지지만 닿지 않는다". 수학의 무한 개념은 현실 세계에서도 유효한가?

실생활 맥락

나노입자 광사태 현상

광자 1개를 나노 물질에 쪼였을 때 각 단계에서 2배씩 증폭된다. x단계 광자 수 y:

x (단계)012345
y (광자 수)12481632
y = 2ˣ   ← x단계 광자 수

x에 대해 2ˣ의 값이 하나로 정해지므로 y = 2ˣ는 함수이다.

지수함수 최댓값·최솟값 원리

a > 1: x 클수록 y 큼 → 정의역의 최대 x에서 최댓값
0 < a < 1: x 클수록 y 작음 → 정의역의 최소 x에서 최댓값
TOK 포인트

지수함수로 인구 성장을 예측한다(맬서스 모델). 수학 모델이 현실을 완전히 설명하지 못할 때, 그 모델은 여전히 가치 있는가?

2
문제 2★☆☆

(1) y=3ˣ    (2) y=(⅓)ˣ
▼
풀이

풀이

▼

SECTION 02

로그함수

y = log_a x (a > 0, a ≠ 1) — 뜻, 그래프, 성질

정의

로그함수의 뜻

지수함수 y=aˣ은 일대일함수이므로 역함수가 존재한다.

y=aˣ에서 로그의 정의에 의해 x=log_a y
x와 y를 바꾸면: y = log_a x (a>0, a≠1) 이를 a를 밑으로 하는 로그함수라 한다.
정의역: 양의 실수 전체 (x>0)
치역: 실수 전체
TOK

로그함수는 지수함수의 "역"이다. 수학에서 '역(inverse)'이라는 개념은 어떤 조건에서 항상 존재하는가?

지수함수 ↔ 로그함수 비교

y = aˣ 와 y = log_a x

항목y = aˣ (지수함수)y = log_a x (로그함수)
정의역실수 전체양의 실수 전체
치역양의 실수 전체실수 전체
기준점(0, 1)(1, 0)
점근선x축 (y=0)y축 (x=0)
a>1 증감증가증가
0<a<1 증감감소감소
관계서로 역함수 — y=x에 대해 대칭

두 그래프는 y=x에 대해 대칭

성질

로그함수
y = log_a x 의 성질

  • 1정의역: 양의 실수 전체
    치역: 실수 전체
  • 2일대일함수
  • 3a>1: x 단조증가 → y 단조증가
    0<a<1: x 단조증가 → y 단조감소
  • 4점 (1, 0) 통과
    y축이 점근선 (x=0)
y = log_a 1 = 0    y = log_a a = 1
TOK

자외선 차단제 SPF, 컴퓨터 저장 공간 — 로그로 나타낸다. 왜 로그가 일상에 자주 등장하는가?

그래프 변환

이동과 점근선

y = log_a (x − p) + q
y = log_a x를 x축으로 +p, y축으로 +q 이동
점근선: x = p (y축도 같이 이동!)
진수 조건: 로그의 진수는 반드시 양수
log_a f(x)에서 f(x) > 0 을 먼저 확인!
함수이동점근선
y=log₂(x+2)x축으로 -2x=−2
y=log_(⅓)(x−2)+1x로 +2, y로 +1x=2
y=−log₃xx축 대칭x=0
y=−log₅x−1x축 대칭 후 y로 −1x=0
TOK 포인트

y=log₂x², y=2log₂x, y=2log₂|x|는 같은 함수인가? 정의역이 다르면 다른 함수다 — 수학에서 "같다"는 것의 기준은 무엇인가?

지수·로그함수의 활용

방정식 · 부등식 · 실생활 문제

핵심 원리

지수
방정식·부등식

방정식:
a>0, a≠1 일 때
aˣ¹ = aˣ² ⟺ x₁ = x₂
부등식 — 밑 주의!
a>1: aˣ¹ < aˣ² ⟺ x₁ < x₂
0<a<1: aˣ¹ < aˣ² ⟺ x₁ > x₂
TOK

밑이 1보다 크면 부등호 유지, 1보다 작으면 뒤집힌다. 직관적으로 설명할 수 있는가?

핵심 원리

로그
방정식·부등식

방정식:
log_a x = b ⟺ x = aᵇ
log_a x₁ = log_a x₂ ⟺ x₁ = x₂
⚠ 진수 조건 확인!
log_a f(x)를 풀 때 반드시 f(x) > 0 확인
x값 구한 후 대입 검산 필수
부등식 — 밑 주의!
a>1: log_a x₁ < log_a x₂ ⟺ x₁ < x₂
0<a<1: log_a x₁ < log_a x₂ ⟺ x₁ > x₂
TOK 포인트

진수 조건을 놓치면 잘못된 해를 얻는다. "조건"을 명시하는 것은 왜 수학의 윤리적 의무이기도 한가?

(1) 2ˣ=64   (2) (⅓)^(x−4)=27   (3) 4^(x+3)=8^(x−1)   (4) 25^(2−x)=(1/125)^(x−1)
▼

풀이

(1) 2ˣ=2⁶ → x=6 (2) (⅓)^(x−4)=3^(−(x−4))=3^(−3) → x=1 (3) 2^(2x+6)=2^(3x−3) → 2x+6=3x−3 → x=9 (4) 5^(2(2−x))=5^(−3(x−1)) → 4−2x=−3x+3 → x=−1

▼

풀이

로1
로그 문제 3★★☆  다음 방정식을 푸시오.

풀이

로2

▼

풀이

로3
로그 문제 5·6★★☆  다음 부등식을 푸시오.
(1) log₂(x−3)<2   (2) log_(⅓)(2x−1)≥log_(⅓)(3x+2)   (3) log₃(x+1)≥3   (4) log_(½)(x+5)<log_(½)(4x−7)
