SECTION 01

지수함수

y = aˣ (a > 0, a ≠ 1) — 뜻, 그래프, 성질

정의

Exponential Functions의 뜻

a > 0, a ≠ 1 일 때, 실수 전체의 집합을 정의역으로 하는 함수 y = aˣ 를 a를 밑으로 하는 Exponential Functions라 한다.
  • 1정의역: 실수 전체 / 치역: 양의 실수 전체
  • 2일대일함수
  • 3a>1 → x 증가 시 y 증가
    0<a<1 → x 증가 시 y 감소
  • 4점 (0, 1) 통과
    점근선: x축 (y = 0)
TOK

a=1이면 y=1 (상수함수). 왜 a≠1 이어야 하는가? 수학에서 "흥미로운 함수"의 조건은?

그래프 비교

a > 1 vs 0 < a < 1

항목a > 1 예: y = 2ˣ0 < a < 1 예: y = (½)ˣ
증감증가감소
기준점(0, 1) 통과
점근선x축 (y = 0)
Range양의 All reals
x → +∞y → +∞y → 0
x → −∞y → 0y → +∞

두 그래프는 y축에 대해 대칭  |  y=(½)ˣ = 2⁻ˣ

그래프 변환

Exponential Functions의
그래프 변환

y = f(x) → y = f(x - a) + b
Shift a units horizontally, b units vertically
y = 2^(x+1) − 3

← y = 2ˣ를 x축으로 −1, y축으로 −3 Shift
Asymptote: y = −3

y = (⅓)^(x−1) + 2

← y = (⅓)ˣ를 x축으로 +1, y축으로 +2 Shift
Asymptote: y = 2

TOK

그래프가 x축에 "한없이 가까워지지만 닿지 않는다". 수학의 무한 개념은 현실 세계에서도 유효한가?

실생활 맥락

나노입자 광사태 현상

광자 1개를 나노 물질에 쪼였을 때 각 단계에서 2배씩 증폭된다. x단계 광자 수 y:

x (단계)012345
y (광자 수)12481632
y = 2ˣ   ← x단계 광자 수

x에 대해 2ˣ의 값이 하나로 정해지므로 y = 2ˣ는 함수이다.

Exponential Functions 최댓값·최솟값 원리

a > 1: x 클수록 y 큼 → 정의역의 최대 x에서 최댓값
0 < a < 1: x 클수록 y 작음 → 정의역의 최소 x에서 최댓값
TOK 포인트

Exponential Functions로 인구 성장을 예측한다(맬서스 모델). 수학 모델이 현실을 완전히 설명하지 못할 때, 그 모델은 여전히 가치 있는가?

SECTION 02

로그함수

y = log_a x (a > 0, a ≠ 1) — 뜻, 그래프, 성질

정의

Logarithmic Functions의 뜻

Exponential Functions y=aˣ은 일대일Function이므로 역Function가 존재한다.

y=aˣ에서 로그의 정의에 의해 x=log_a y
x와 y를 바꾸면: y = log_a x (a>0, a≠1) 이를 a를 밑으로 하는 Logarithmic Functions라 한다.
Domain: 양의 All reals (x>0)
Range: All reals
TOK

Logarithmic Functions는 Exponential Functions의 "역"이다. 수학에서 '역(inverse)'이라는 개념은 어떤 조건에서 항상 존재하는가?

Exponential Functions ↔ Logarithmic Functions 비교

y = aˣ 와 y = log_a x

항목y = aˣ (Exponential Functions)y = log_a x (Logarithmic Functions)
정의역실수 전체양의 All reals
Range양의 All reals실수 전체
기준점(0, 1)(1, 0)
점근선x축 (y=0)y축 (x=0)
a>1 증감증가증가
0<a<1 증감감소감소
관계서로 역함수 — y=x에 대해 대칭

두 그래프는 y=x에 대해 대칭

성질

Logarithmic Functions
y = log_a x 의 성질

  • 1Domain: 양의 All reals
    Range: 실수 전체
  • 2일대일함수
  • 3a>1: x Increasing → y Increasing
    0<a<1: x Increasing → y Decreasing
  • 4점 (1, 0) 통과
    y축이 Asymptote (x=0)
y = log_a 1 = 0    y = log_a a = 1
TOK

자외선 차단제 SPF, 컴퓨터 저장 공간 — 로그로 나타낸다. 왜 로그가 일상에 자주 등장하는가?

그래프 변환

Shift과 Asymptote

y = log_a (x − p) + q
y = log_a x를 x축으로 +p, y축으로 +q Shift
Asymptote: x = p (y축도 같이 Shift!)
진수 조건: 로그의 진수는 반드시 양수
log_a f(x)에서 f(x) > 0 을 먼저 확인!
함수이동점근선
y=log₂(x+2)x축으로 -2x=−2
y=log_(⅓)(x−2)+1x로 +2, y로 +1x=2
y=−log₃xx축 대칭x=0
y=−log₅x−1reflect over x-axis 후 y로 −1x=0
TOK 포인트

y=log₂x², y=2log₂x, y=2log₂|x|는 같은 Function인가? Domain이 다르면 다른 Function다 — 수학에서 "같다"는 것의 기준은 무엇인가?

지수
방정식·부등식

방정식:
a>0, a≠1 일 때
aˣ¹ = aˣ² ⟺ x₁ = x₂
부등식 — 밑 주의!
a>1: aˣ¹ < aˣ² ⟺ x₁ < x₂
0<a<1: aˣ¹ < aˣ² ⟺ x₁ > x₂
TOK

로그
방정식·부등식

방정식:
log_a x = b ⟺ x = aᵇ
log_a x₁ = log_a x₂ ⟺ x₁ = x₂
⚠ 진수 조건 확인!

Quadratic y=f(x) passes through (0,4), (2,0), (4,4). Find the number of integers x satisfying log_(¼)f(x) ≥ −1.
▼
정답

4 integers (x = 0, 1, 5, 6)

풀이

f(x) is quadratic with vertex at x=2, f(2)=0: f(x) = a(x−2)². f(0)=4 → 4a=4 → a=1 → f(x)=(x−2)² log_(¼)f(x) ≥ −1 = log_(¼)4 → f(x) ≤ 4 (base ¼<1 reverses inequality). Also f(x)>0 (argument condition): (x−2)²≠0 → x≠2 (x−2)² ≤ 4 → −2 ≤ x−2 ≤ 2 → 0 ≤ x ≤ 4, but x≠2 Integer solutions: x = 0, 1, 3, 4 → 4 integers

Newton's Cooling Law (창의 Up 프로젝트, p.62)★★★
T = T_s + (T₀ − T_s)×10^(−t/10), where T₀=100°C, T_s=room temperature.
Measure T_s, then compare measured vs. calculated temperatures every 5 min for 30 min.
▼
Example (T_s = 25°C)

T(0) = 25 + 75×10⁰ = 100°C T(5) = 25 + 75×10^(−0.5) = 25 + 75×0.3162 ≈ 48.7°C T(10) = 25 + 75×10⁻¹ = 25 + 7.5 = 32.5°C T(30) = 25 + 75×10⁻³ = 25 + 0.075 ≈ 25.1°C ≈ room temp Measured values differ slightly because cooling rate is not perfectly uniform.
